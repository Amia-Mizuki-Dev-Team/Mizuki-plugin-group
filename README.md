# Mizuki-plugin-group

Mizuki Bot 的群聊基础插件。

本仓库用于维护群聊场景中的基础信息整理、群功能入口和群内交互辅助能力。它更适合作为其他插件的配套模块，而不是单独面向普通用户的一键产品。

## 项目定位

`Mizuki-plugin-group` 主要面向 Mizuki Bot 的群聊使用场景。仓库内容应围绕群聊配置、群内提示、群功能状态和基础交互体验展开。

可能包含的内容包括：

- 群聊功能说明
- 群内基础交互逻辑
- 群配置相关辅助代码
- 与帮助、欢迎、发送等插件的协作内容

具体能力请以代码实现为准。
