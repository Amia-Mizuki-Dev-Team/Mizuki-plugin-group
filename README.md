# Mizuki Plugin Group

Mizuki Plugin Group 是开发组内部维护的群聊基础插件，用于整理群聊场景中的基础信息、群功能入口和通用交互能力。

本仓库当前不作为公开通用插件发布。README 仅说明项目定位、职责边界与维护注意事项，不提供完整部署流程、公开配置说明或外部调用方式。

## 项目定位

`Mizuki-plugin-group` 面向 Mizuki Bot 的群聊使用场景，更适合作为其他插件的配套模块，而不是独立的一键产品。

典型职责包括：

- 维护群聊基础功能入口。
- 整理群内提示和基础交互逻辑。
- 协助帮助、欢迎、发送等插件保持一致的群聊体验。
- 为其他业务插件提供稳定的群聊侧基础能力。

## 当前状态

- 维护状态：开发组内部维护中。
- 使用范围：面向现有机器人环境和固定群聊场景，不保证外部环境可直接运行。
- 文档状态：暂不补公开部署说明、完整配置项或调用方式。
- 许可证状态：暂未选择开源许可证。未经明确许可，不建议再分发、商用或用于生产部署。

## 维护注意事项

- 群聊基础插件应优先保证行为稳定，避免频繁改变用户可见入口。
- 与 help、welcome、send 等插件存在协作关系时，应保持命名和状态说明一致。
- 新增功能后应同步更新文档站或帮助索引，避免入口失效。
- 涉及群聊状态、权限或提示消息的改动，应避免影响其他业务插件。
